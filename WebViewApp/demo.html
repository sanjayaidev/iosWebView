About This Demo

This HTML page demonstrates bidirectional communication between JavaScript and native iOS code:

  • WKScriptMessageHandler: Allows JavaScript to send messages to native code using window.webkit.messageHandlers
  • evaluateJavaScript(): Enables native code to execute JavaScript and update the UI

Note: When running in the iOS app, the buttons will communicate with native Swift code. In a regular browser, they'll show simulated responses.
